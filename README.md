## Flujo de trabajo semanal (resumen)

1. Reviso el **calendario maestro** para ver qué toca esta semana.
2. Diseño en Canva usando el **Kit de Marca**.
3. Escribo el caption siguiendo la estructura (Gancho → Valor → CTA).
4. Reviso **SEO, hashtags y geotag** antes de publicar.
5. Marco el post como "Publicado" en el calendario y guardo su ficha en `04-publicaciones/`.
6. Anoto métricas y aprendizajes.

---

## Ritmo de publicación

- **3 publicaciones al feed por semana:** Martes (educativo), Jueves (resultados/proceso), Sábado (marca personal).
- **Stories los días intermedios** (lun, mié, vie) como mínimo.
